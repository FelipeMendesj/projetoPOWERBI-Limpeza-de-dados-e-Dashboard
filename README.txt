
# PROJETO LIMPEZA DE DADOS E DASHBORD INTERATIVA - POWER BI 

- TRATAMENTO DE DADOS

- ANALISE DE DADOS 

- CRIAÇÃO DE DASHBORD

Durante o processo de ETL, foram realizadas as seguintes etapas:

Separação de colunas com dados mistos (ex: quantidade e valor de vendas na mesma coluna)
Correção de formatação de moeda (valores sem casas decimais)
Padronização dos tipos de dados
Organização das colunas para melhor leitura e análise
Remoção de inconsistências e dados incorretos

O dashboard foi desenvolvido com foco em análise de vendas, permitindo visualizar:

Faturamento total
Quantidade de vendas
Evolução das vendas ao longo do tempo
Produtos ou categorias com melhor desempenho
Indicadores gerais para apoio à tomada de decisão

Ferramentas Utilizadas: Excel (fonte de dados)
Power BI (Power Query + Dashboard)
